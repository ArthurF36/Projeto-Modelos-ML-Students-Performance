# Projeto modelos ML: Students Performance

📊 Student Performance Analysis
Este projeto tem como objetivo analisar o desempenho acadêmico de estudantes utilizando técnicas de Machine Learning, explorando tanto modelos de classificação quanto de regressão.

📌 Objetivo
O principal objetivo é prever o desempenho dos alunos com base em características como hábitos de estudo, contexto familiar e atividades extracurriculares.

O problema é abordado de duas formas:

🔹 Classificação → prever a classe de desempenho (GradeClass)
🔹 Regressão → analisar a relação entre tempo de estudo e desempenho (GPA)

📁 Dataset
O conjunto de dados contém:

2392 registros de estudantes
15 variáveis (features)

As variáveis incluem:

Dados demográficos (idade, gênero, etnia)
Contexto familiar (educação e apoio dos pais)
Hábitos de estudo (tempo semanal, faltas, reforço)
Atividades extracurriculares (esporte, música, etc.)
Desempenho acadêmico (GPA e GradeClass)

📎 Fonte dos dados:
Os dados utilizados neste projeto estão disponíveis publicamente no Kaggle pelo autor Rabie El Kharoua.
